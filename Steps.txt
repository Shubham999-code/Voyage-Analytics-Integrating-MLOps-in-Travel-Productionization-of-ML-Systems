Infrastructure as Code

Docker (Dockerfile): A unified multi-purpose Dockerfile to easily containerize and run either the Flask API or Streamlit application.

Kubernetes (kubernetes/): Deployment and Service configurations to scale your API and UI on a K8s cluster.

Airflow (airflow/dags/model_training_dag.py): An automated workflow DAG that uses papermill to schedule and execute your Jupyter notebook periodically.

Jenkins (Jenkinsfile): A complete CI/CD pipeline to test, build the Docker image, push it to Docker Hub, and roll out to Kubernetes.
